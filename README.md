# go-uids



NOT FOR PRODUCTION USE


uuid16r (reverse)

128bits universal unique id lexicographically ordered, url safe, time based with auto clock correction 
and strictly decremental

Sample

    ~~~cM6ajvtg~0smDUwh41l
    ~~~cM6ajuE7~pUB6IO65MW
    ~~~cM6ajsXR~r6WT3ORHvG
    ~~~c4VxsK_o~bUjT6AH7X0

i.e.:
